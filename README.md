Project Description:
Developed a customer segmentation system for an e-commerce platform using unsupervised machine learning (K-Means Clustering) to identify groups of customers with similar purchasing behavior. Performed data cleaning, feature engineering, exploratory data analysis (EDA), feature scaling, and clustering to discover valuable customer segments.
Created new business features such as Age, Total Children, Total Spending, and Customer Membership Duration. Applied clustering techniques to group customers based on demographics, income, spending habits, and purchasing patterns. Evaluated the optimal number of clusters using the Elbow Method and visualized customer groups to generate actionable business insights.
The project helps businesses design targeted marketing campaigns, improve customer retention, identify high-value customers, and increase overall sales performance.
